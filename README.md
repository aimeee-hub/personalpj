
## 商品管理システム

### 概要
このシステムでは商品の在庫管理を行うことができます。  
商品のカテゴリごとの登録から編集、削除まで行うことができ、各商品の在庫を調整管理することができます。

### 主な機能
* ログイン・ログアウト機能
* 商品一覧画面
* 商品の新規登録・編集・削除機能
